Painel

O fluxo segue o pipeline da pesquisa: o modelo ranqueia os suspeitos, o revisor faz a triagem do objeto e, para os confirmados, o sistema faz a análise documental do rito (baixa o edital/TR). Cada triagem retroalimenta o modelo.

Triagem do objeto

Etapa 1 (rápida): pelo objeto do contrato, decida se é engenharia. Ao confirmar, o caso vai para a fila de Análise de rito. Sua decisão ensina o modelo.

Análise de rito

Etapa 2 (posterior): só para os contratos confirmados como engenharia. O sistema baixa o edital / Termo de Referência da licitação e procura os marcadores do rito (ART/CREA, projeto básico, ABNT, planilha orçamentária…). Você dá o veredito final.
